# Kubernetes Security Prototype

This folder documents a Kubernetes migration path for the OpenCTI deployment.

The goal is not to claim a full production Kubernetes deployment, but to demonstrate how the Docker Compose MVP could evolve into a more mature Kubernetes-based architecture.

Included security concepts:

- Dedicated namespace
- Dedicated ServiceAccount
- Namespace-scoped RBAC
- NetworkPolicy examples
- SecurityContext examples
- Secrets templates
- k3s/DOKS migration notes
